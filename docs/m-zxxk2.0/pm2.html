1. pm2常用命令

1.1. 启动服务

pm2 start appName|id|all

1.2. 停止服务

pm2 stop appName|id|all

1.3. 重载服务

该命令常用，因为可以实现0s无缝重启，类似nginx reload
pm2 reload appName|id|all

1.4. 重启服务

该命令不常用，一般用pm2 reload app
pm2 restart appName|id|all

1.5. 删除服务

pm2 delete appName|id|all

1.6. 列出所有进程列表

常用
pm2 list

1.7. 查看所有进程的状态

pm2 status

1.8. 监控

常用
pm2 monit

1.9. 显示某个进程的详情

pm2 show appName|id
